# Glosario (W01)

- **Raw / Bronze**: capa de datos tal como llegan desde la fuente, sin transformaciones semánticas profundas.
- **Sanity check**: verificación rápida para detectar errores obvios (nulos, rangos inválidos, duplicados, etc.).
- **Trazabilidad**: capacidad de rastrear de dónde viene un dato y cómo fue procesado.
- **SHA-256**: huella criptográfica del archivo para detectar cambios invisibles.
